## 一、项目目标

在用户**无法访问国外网站**的约束下，**本地完整走通**地区化今天历史档案站的全链路流程：

1. **数据获取**：Mock Wikipedia OnThisDay API
2. **AI 管道**：Mock Claude/GPT-4o/Gemini（地区化重写 + 翻译 + 配图）
3. **内容审核**：Mock 审核接口（中立性 + 合规性）
4. **站点生成**：Hugo 10 万页面（10 语种 × 30 地区 × 366 天）
5. **本地预览**：`http://localhost:1313` 看到可浏览的网站
6. **生产迁移**：Phase 4 用户提供网络后切换到真实 API + Cloudflare Pages

## 五、关键约束

1. **网络隔离**：本地开发阶段（Phase 0-3）不访问任何国外网站
2. **先走通本地**：Phase 0-3 必须在本地完整跑通
3. **看到网站内容**：Phase 3 末用户可在 `http://localhost:1313` 看到网站
4. **每步独立文件**：每个步骤对应独立 `.md` 文件
5. **最小化修改**：不修改与本任务无关内容
6. **禁止 git 恢复**：不从 git 上直接恢复内容
7. **逐个修改**：禁止写代码执行替换方式修改程序
