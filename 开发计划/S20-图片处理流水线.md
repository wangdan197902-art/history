> 阶段：Phase 3 - Hugo 集成与 10 万页构建

## 一、步骤概述

完善图片处理流水线：Hugo Pipes 自动生成 WebP 响应式图片（400w/800w 双源），通过 `<picture>` 标签呈现，srcset 自适应。单张图 < 100KB，支持懒加载。

## 二、任务清单

### 2.1 图片处理 partial

文件：`site/layouts/partials/responsive-image.html`

```html
{{/* 用法: {{ partial "responsive-image.html" (dict "src" .image_url "alt" .image_alt "caption" .image_caption) }} */}}
{{ $src := .src }}
{{ $alt := .alt | default "历史图片" }}
{{ $caption := .caption }}

{{ if $src }}
  {{ $resource := resources.GetRemote $src }}
  {{ if $resource }}
    {{ $image400 := $resource.Resize "400x webp q80" }}
    {{ $image800 := $resource.Resize "800x webp q80" }}
    <figure class="responsive-image">
      <picture>
        <source media="(max-width: 480px)" srcset="{{ $image400.RelPermalink }}">
        <source media="(min-width: 481px)" srcset="{{ $image800.RelPermalink }}">
        <img src="{{ $image800.RelPermalink }}" alt="{{ $alt }}" loading="lazy" width="{{ $image800.Width }}" height="{{ $image800.Height }}">
      </picture>
      {{ if $caption }}
        <figcaption>{{ $caption }}</figcaption>
      {{ end }}
    </figure>
  {{ else }}
    {{/* 远程图片失败时降级到占位图 */}}
    <figure class="responsive-image">
      <img src="/images/placeholder.svg" alt="{{ $alt }}" loading="lazy">
      <figcaption>{{ $caption }}</figcaption>
    </figure>
  {{ end }}
{{ end }}
```

### 2.6 Hugo 图片缓存配置

文件：`site/hugo.toml`（增强）

```toml
[imaging]
  quality = 80
  resampleFilter = "Lanczos"
  hint = "photo"

[imaging.exif]
  disableDate = false
  disableLatLong = true
  includeFields = "Artist|Copyright"
```

## 六、风险提示

| 风险 | 等级 | 应对 |
|------|------|------|
| Hugo Pipes 远程图片失败 | 中 | 降级到占位图 |
| WebP 体积超限 | 低 | 自动降低 quality |
| Hugo 构建慢 | 中 | --cacheDir 持久化 |

## 七、性能预算

| 指标 | 目标值 | 测量方法 |
|------|--------|---------|
| 300 张图 WebP 转换 | < 60s | `time` |
| 单图体积 | < 100 KB | `ls -la` |
| `<picture>` 响应式标签 | 100% | `grep` |

## 八、测试要求

- WebP 双源（400w/800w）生成
- `<picture>` 标签正确
- 单图 < 100KB
- 懒加载 `loading="lazy"`
- 占位图降级正常
