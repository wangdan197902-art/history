## 1. 项目总览

### 1.1 项目愿景

构建一个**地区化的"今天历史"档案站**，覆盖 30 地区 × 366 天 × 10 语种 ≈ 10 万页面，通过每日更新形成天然周期性回访流量，实现低成本高回报的内容矩阵站点。

### 1.2 核心目标

| 维度 | 12 个月目标 | 36 个月目标 |
|------|-----------|------------|
| 页面数 | 110,000 | 500,000+ |
| 月 UV | 50,000-150,000 | 200,000-500,000 |
| 月收入 | $1,500-3,000 | $5,000-20,000 |
| 月成本 | $300 | $2,300 |
| 月净利润 | $1,200-2,700 | $2,700-17,700 |
| 邮件订阅 | 5,000+ | 30,000+ |
| 自动化率 | 95% | 98% |
| 人工投入 | 5-10 小时/周 | 5-10 小时/周 |

### 1.3 关键成功因素

1. **每日更新机制**：天然回访 + 邮件订阅留存
2. **30 地区 × 10 语种覆盖**：广度壁垒，3 年累积难追
3. **AI 叙事改写**：避免抄袭 Wikipedia，保证原创性
4. **政治中立性**：敏感事件 AI 识别 + 人工抽检
5. **联盟 + 邮件双变现**：AdSense 前用 Amazon + 邮件赞助

### 1.4 项目时间线总览

```
2026
├── 7月  ┌─ Phase 0 (Week 0): 准备期
│       │  └── 基础设施 + 项目搭建
│       │
├── 8月  ┌─ Phase 1 (Week 1-4): MVP
│       │  ├── Week 1: 核心架构
│       │  ├── Week 2: 事件采集管道
│       │  ├── Week 3: 内容生产 + 发布
│       │  └── Week 4: 变现接入
│       │
├── 9月-12月 ┌─ Phase 2 (Month 2-6): 增长期
│           │  ├── 多语种扩展
│           │  ├── SEO 冲刺
│           │  └── AdSense 申请
│           │
├── 2027 1-6月 ┌─ Phase 3 (Month 7-12): 成熟期
│              │  ├── 年度循环优化
│              │  ├── 邮件订阅扩展
│              │  └── API 上线
│              │
└── 2027 7月 - 2029 ┌─ Phase 4 (Year 2-3): 扩展期
                    │  ├── 多站点矩阵
                    │  ├── 媒体合作
                    │  └── 白标授权
```
