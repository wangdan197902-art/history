## 1. AI 闭环总览

### 1.1 AI 在项目中的角色

本项目的核心特征是 **AI 参与度 ≥ 95%**：从选题、采集、地区化、翻译、配图、审核、发布、SEO 到反馈优化，每一个环节均有 AI 模型直接产出可发布内容，人工仅承担 5% 的抽检与异常处理。

> ⚠️ **中立性最高优先级原则**：所有 AI 生成内容必须遵守历史事实中立性原则。涉及战争、政治、宗教、领土争议的事件，AI 输出必须保持多方视角平衡，详见第 6、7 章。

### 2.3 模型选择决策树

```python
def select_model(task: str, lang: str, sensitivity: int) -> str:
    """根据任务类型选择 AI 模型。

    Args:
        task: 任务类型 (regionalize/translate/audit/illustrate/seo)
        lang: 目标语言代码
        sensitivity: 敏感等级 L1-L5

    Returns:
        模型名称字符串
    """
    if task == "regionalize":
        # 地区化重写：Claude 主力（中立性强、叙事能力好）
        return "claude-3-5-sonnet-20241022"
    elif task == "translate":
        # 翻译：GPT-4o 多语种能力最强
        return "gpt-4o-2024-08-06"
    elif task == "audit":
        # 审核：敏感事件 L3+ 必须 Claude 复核（中立性更可靠）
        if sensitivity >= 3:
            return "claude-3-5-sonnet-20241022"
        return "gpt-4o-2024-08-06"
    elif task == "illustrate":
        # 配图描述：Gemini Flash 低成本
        return "gemini-1.5-flash-002"
    elif task == "seo":
        # SEO 元数据：GPT-4o 结构化输出
        return "gpt-4o-2024-08-06"
    else:
        raise ValueError(f"未知任务类型: {task}")
```

### 3.3 端到端失败恢复

```python
# scripts/orchestrator_recovery.py
"""端到端失败恢复：根据 /data/state/daily_{YYYYMMDD}.json 续跑。"""
import json
from pathlib import Path
from datetime import datetime

def resume_pipeline(target_date: str):
    """根据状态文件续跑未完成的阶段。

    Args:
        target_date: YYYYMMDD 格式日期
    """
    state_file = Path(f"/data/state/daily_{target_date}.json")
    if not state_file.exists():
        print(f"[RESTART] 状态文件不存在，从头开始: {target_date}")
        return run_full_pipeline(target_date)

    state = json.loads(state_file.read_text())
    stages = ["topic", "fetch", "regionalize", "translate",
              "illustrate", "audit", "publish", "seo"]

    for stage in stages:
        if not state.get(stage, {}).get("completed", False):
            print(f"[RESUME] 从 {stage} 阶段续跑: {target_date}")
            return run_from_stage(stage, target_date, state)

    print(f"[DONE] 当日全部完成: {target_date}")
```

> 所有 Prompt 模板均为完整可用版本，使用 Jinja2 风格变量 `{{ var }}`。所有模板强制包含中立性指令与敏感事件处理规则。

### 4.1 选题 Prompt（AI-TOPIC）

```text
你是资深历史编辑，负责为"地区化今天历史"档案站筛选事件。

【日期】{{ date }}（{{ weekday }}）

【候选事件池（来自 Wikipedia OnThisDay）】
{{ events_json }}

【目标地区清单】
{{ countries | join(', ') }}

【筛选规则（严格）】
1. 事件必须满足以下任一条件才入选：
   a. 事件发生在该国当前领土范围内
   b. 事件主人公为该国公民或政府
   c. 事件对该国政治/经济/文化有直接重大影响
2. 排除条件：
   - 纯体育赛事（除非为奥运会/世界杯级别）
   - 娱乐圈八卦
   - 未被多源验证的传闻
3. 每地区筛选 5-15 条事件，按重要性降序
4. **中立性强制要求**：
   - 涉及领土争议的事件，标注【领土争议】标签
   - 涉及政治立场的事件，标注【政治敏感】标签
   - 涉及宗教冲突的事件，标注【宗教敏感】标签
   - 不得在筛选阶段预设立场

【输出格式（严格 JSON）】
{
  "date": "{{ date }}",
  "countries": {
    "us": [
      {
        "event_id": "evt_001",
        "year": 1969,
        "headline": "Apollo 11 landed on the Moon",
        "summary": "Neil Armstrong and Buzz Aldrin became the first humans to walk on the Moon.",
        "sensitivity_level": 1,
        "tags": ["science", "space"],
        "sources": ["https://en.wikipedia.org/wiki/Apollo_11"]
      }
    ]
  },
  "skipped": [
    {"event": "...", "reason": "未通过多源验证"}
  ]
}

【严禁】
- 不要输出任何 JSON 之外的内容
- 不要在 summary 中加入主观评价
- 不要预设立场（如"胜利/失败/正义/邪恶"）
```

### 4.2 地区化重写 Prompt（AI-REGIONALIZE）

```text
你是历史编辑，为{{ country_name }}（ISO: {{ country_code }}）生成"今天历史"页面。

【日期】{{ date }}
【地区相关事件】
{{ events_json }}

【重写规则（严格）】
1. 每事件改写为 80-200 字叙事，禁止直接复制 Wikipedia 原文
2. 按重要性降序排列
3. 必须包含"该地区视角"：事件对该地区的影响/意义
4. 时间格式统一为"YYYY 年 M 月 D 日"
5. 人名/地名使用该地区官方语言习惯（如：德国事件用"柏林"而非"Berlin"）

【中立性强制规则（最高优先级）】
1. **战争事件**：使用中立词汇（"交战""冲突"），不得使用"侵略/解放/正义战争"等带立场词
2. **政治事件**：呈现多方立场，不预设立场
```

> 本节为端到端编排脚本，与 03_内容获取与生产闭环.md 中的分模块脚本形成"总-分"关系。本脚本调用各分模块并完成全链路。

### 5.1 端到端编排主脚本

```python
# scripts/orchestrator.py
"""地区化今天历史档案站 - AI 端到端编排主脚本。

每日 UTC 01:00 由 GitHub Actions 触发，完成 10 阶段闭环。
任何阶段失败均会写入状态文件，下次运行可续跑。

Usage:
    python scripts/orchestrator.py --date 2026-07-18
    python scripts/orchestrator.py --date 2026-07-18 --resume
"""
from __future__ import annotations

import argparse
import json
import logging
import sys
import time
from datetime import datetime, timezone
from pathlib import Path

from ai.clients import ClaudeClient, GPT4oClient, GeminiClient
from ai.cost_tracker import CostTracker
from ai.sensitivity_classifier import classify_sensitivity
from pipeline.audit import audit_content
from pipeline.fetch import fetch_events
from pipeline.illustrate import illustrate_events
from pipeline.publish import publish_pages
from pipeline.regionalize import regionalize_events
from pipeline.seo import generate_seo_metadata
from pipeline.translate import translate_content
from pipeline.topic import select_topics
from pipeline.monitor import collect_metrics
from pipeline.feedback import generate_feedback

logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
)
log = logging.getLogger("orchestrator")

# 30 地区 ISO 清单（与 01_技术架构体系.md 一致）
COUNTRIES = [
    "us", "gb", "de", "fr", "es", "it", "ru", "cn", "jp", "kr",
    "in", "br", "mx", "ar", "ca", "au", "nz", "za", "eg", "tr",
    "gr", "pl", "ua", "se", "no", "nl", "be", "pt", "ie", "at",
]

# 10 语种代码
LANGUAGES = ["en", "zh", "es", "fr", "de", "it", "pt", "ru", "ja", "ko"]

STATE_DIR = Path("/data/state")
STATE_DIR.mkdir(parents=True, exist_ok=True)


def load_state(date: str) -> dict:
    """加载当日状态文件，用于续跑。"""
    state_file = STATE_DIR / f"daily_{date}.json"
    if state_file.exists():
        return json.loads(state_file.read_text())
    return {
        "date": date,
        "started_at": datetime.now(timezone.utc).isoformat(),
        "stages": {},
    }


def save_state(date: str, state: dict) -> None:
    """保存当日状态文件。"""
    state_file = STATE_DIR / f"daily_{date}.json"
    state_file.write_text(json.dumps(state, indent=2, ensure_ascii=False))


def mark_stage(state: dict, stage: str, status: str, **extra) -> None:
    """标记某阶段状态。"""
    state["stages"][stage] = {
        "status": status,  # running/completed/failed/skipped
        "updated_at": datetime.now(timezone.utc).isoformat(),
        **extra,
    }
    save_state(state["date"], state)


def run_stage(state: dict, stage: str, fn, *args, **kwargs):
    """运行某阶段，失败时记录状态并抛出。"""
    if state["stages"].get(stage, {}).get("status") == "completed":
        log.info(f"[SKIP] {stage} 已完成")
        return None
    mark_stage(state, stage, "running")
    try:
        result = fn(*args, **kwargs)
        mark_stage(state, stage, "completed")
        return result
    except Exception as e:
        log.exception(f"[FAIL] {stage} 失败: {e}")
        mark_stage(state, stage, "failed", error=str(e))
        raise


def run_full_pipeline(target_date: str, resume: bool = False) -> dict:
    """运行端到端闭环。

    Args:
        target_date: YYYYMMDD 格式日期
        resume: 是否从状态文件续跑

    Returns:
        当日状态字典
    """
    state = load_state(target_date) if resume else {"date": target_date, "stages": {}}
    state["date"] = target_date
    cost_tracker = CostTracker(daily_budget_usd=2.0)

    log.info(f"=== 开始处理 {target_date} ===")

    # Stage 1: 选题
    claude = ClaudeClient(cost_tracker=cost_tracker)
    topics = run_stage(
        state, "topic",
        select_topics,
        date=target_date, countries=COUNTRIES, model_client=claude,
    )

    # Stage 2: 采集
    gpt4o = GPT4oClient(cost_tracker=cost_tracker)
    raw_events = run_stage(
        state, "fetch",
        fetch_events,
        date=target_date, topics=topics, model_client=gpt4o,
    )

    # Stage 3: 地区化（30 地区并行）
    regional_data = run_stage(
        state, "regionalize",
        regionalize_events,
        date=target_date, raw_events=raw_events,
        countries=COUNTRIES, model_client=claude,
    )

    # Stage 3.5: 敏感事件分级
    for country, data in regional_data.items():
        for event in data.get("events", []):
            event["sensitivity_level"] = classify_sensitivity(event, country)

    # Stage 4: 翻译（10 语种并行）
    translated = run_stage(
        state, "translate",
        translate_content,
        regional_data=regional_data,
        languages=LANGUAGES, model_client=gpt4o,
    )

    # Stage 5: 配图
    gemini = GeminiClient(cost_tracker=cost_tracker)
    illustrated = run_stage(
        state, "illustrate",
        illustrate_events,
        regional_data=regional_data, model_client=gemini,
    )

    # Stage 6: 审核（L3+ 必须人工）
    audit_results = run_stage(
        state, "audit",
        audit_content,
        translated=translated, illustrated=illustrated,
        model_client_primary=gpt4o,
        model_client_neutrality=claude,
    )

    # 阻塞 L4+ 事件
    blocked = [r for r in audit_results if r["audit_result"] == "blocked"]
    if blocked:
        log.error(f"[BLOCKED] {len(blocked)} 个事件被阻塞，需人工处理")
        for b in blocked:
            log.error(f"  - {b['country']}/{b['date']}: {b.get('sensitivity', {})}")

    # Stage 7: 发布
    run_stage(
        state, "publish",
        publish_pages,
        translated=translated, illustrated=illustrated,
        audit_results=audit_results,
    )

    # Stage 8: SEO
    run_stage(
        state, "seo",
        generate_seo_metadata,
        translated=translated, model_client=gpt4o,
    )

    # Stage 9: 监控（UTC 23:00 由独立 cron 触发，此处仅记录准备）
    mark_stage(state, "monitor", "scheduled", trigger_at="23:00 UTC")

    # 输出成本报告
    cost_report = cost_tracker.report()
    state["cost"] = cost_report
    log.info(f"=== 当日成本: ${cost_report['total_usd']:.4f} ===")

    save_state(target_date, state)
    return state


def main():
    parser = argparse.ArgumentParser(description="AI 端到端编排")
    parser.add_argument("--date", required=True, help="YYYYMMDD 格式日期")
    parser.add_argument("--resume", action="store_true", help="从状态文件续跑")
    args = parser.parse_args()

    try:
        state = run_full_pipeline(args.date, resume=args.resume)
        completed = sum(
            1 for s in state["stages"].values()
            if s.get("status") == "completed"
        )
        log.info(f"=== 完成 {args.date}: {completed} 阶段成功 ===")
        sys.exit(0)
    except Exception as e:
        log.exception(f"=== 端到端失败: {e} ===")
        sys.exit(1)


if __name__ == "__main__":
    main()
```

```python
# scripts/ai/clients.py
"""AI 模型客户端统一封装，支持成本追踪与降级。"""
from __future__ import annotations

import logging
import os
import time
from typing import Any

import anthropic
import google.generativeai as genai
import openai
from tenacity import retry, stop_after_attempt, wait_exponential

from ai.cost_tracker import CostTracker

log = logging.getLogger(__name__)

# 模型定价（USD per 1M tokens，2026-07 参考）
PRICING = {
    "claude-3-5-sonnet-20241022": {"input": 3.0, "output": 15.0},
    "gpt-4o-2024-08-06": {"input": 2.5, "output": 10.0},
    "gpt-4o-mini-2024-07-18": {"input": 0.15, "output": 0.6},
    "gemini-1.5-flash-002": {"input": 0.075, "output": 0.3},
    "gemini-1.5-pro-002": {"input": 1.25, "output": 5.0},
}


class BaseClient:
    """AI 客户端基类，统一接口与成本追踪。"""

    def __init__(self, model: str, cost_tracker: CostTracker):
        self.model = model
        self.cost_tracker = cost_tracker

    def _record_cost(self, input_tokens: int, output_tokens: int) -> None:
        price = PRICING.get(self.model, {"input": 0, "output": 0})
        cost = (
            input_tokens * price["input"] / 1_000_000
            + output_tokens * price["output"] / 1_000_000
        )
        self.cost_tracker.add(self.model, cost, input_tokens, output_tokens)
        log.debug(
            f"[COST] {self.model}: in={input_tokens} out={output_tokens} "
            f"cost=${cost:.4f}"
        )

    def chat(self, prompt: str, **kwargs) -> str:
        raise NotImplementedError


class ClaudeClient(BaseClient):
    """Anthropic Claude 客户端。"""

    def __init__(self, cost_tracker: CostTracker, model: str = "claude-3-5-sonnet-20241022"):
        super().__init__(model, cost_tracker)
        self.client = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])

    @retry(stop=stop_after_attempt(3), wait=wait_exponential(min=10, max=60))
    def chat(self, prompt: str, max_tokens: int = 4096, **kwargs) -> str:
        start = time.time()
        response = self.client.messages.create(
            model=self.model,
            max_tokens=max_tokens,
            messages=[{"role": "user", "content": prompt}],
            **kwargs,
        )
        elapsed = time.time() - start
        self._record_cost(
            response.usage.input_tokens, response.usage.output_tokens
        )
        log.info(f"[CLAUDE] {elapsed:.1f}s")
        return response.content[0].text


class GPT4oClient(BaseClient):
    """OpenAI GPT-4o 客户端，支持 JSON 模式。"""

    def __init__(self, cost_tracker: CostTracker, model: str = "gpt-4o-2024-08-06"):
        super().__init__(model, cost_tracker)
        self.client = openai.OpenAI(api_key=os.environ["OPENAI_API_KEY"])

    @retry(stop=stop_after_attempt(3), wait=wait_exponential(min=10, max=60))
    def chat(self, prompt: str, json_mode: bool = False, **kwargs) -> str:
        start = time.time()
        params = {
            "model": self.model,
            "messages": [{"role": "user", "content": prompt}],
            **kwargs,
        }
        if json_mode:
            params["response_format"] = {"type": "json_object"}
        response = self.client.chat.completions.create(**params)
        elapsed = time.time() - start
        self._record_cost(
            response.usage.prompt_tokens, response.usage.completion_tokens
        )
        log.info(f"[GPT-4o] {elapsed:.1f}s")
        return response.choices[0].message.content


class GeminiClient(BaseClient):
    """Google Gemini 客户端。"""

    def __init__(self, cost_tracker: CostTracker, model: str = "gemini-1.5-flash-002"):
        super().__init__(model, cost_tracker)
        genai.configure(api_key=os.environ["GOOGLE_API_KEY"])
        self.gen_model = genai.GenerativeModel(model)

    @retry(stop=stop_after_attempt(3), wait=wait_exponential(min=10, max=60))
    def chat(self, prompt: str, **kwargs) -> str:
        start = time.time()
        response = self.gen_model.generate_content(prompt, **kwargs)
        elapsed = time.time() - start
        # Gemini 不直接返回 token 数，使用估算
        input_tokens = len(prompt) // 4
        output_tokens = len(response.text) // 4
        self._record_cost(input_tokens, output_tokens)
        log.info(f"[GEMINI] {elapsed:.1f}s")
        return response.text
```

```python
# scripts/ai/cost_tracker.py
"""AI 调用成本追踪与预算控制。"""
from __future__ import annotations

import json
import logging
from datetime import datetime, timezone
from pathlib import Path

log = logging.getLogger(__name__)


class CostTracker:
    """按模型记录每日调用成本，超出预算时抛出异常。"""

    def __init__(self, daily_budget_usd: float = 2.0):
        self.daily_budget = daily_budget_usd
        self.records: list[dict] = []
        self.total_usd = 0.0

    def add(
        self, model: str, cost_usd: float,
        input_tokens: int = 0, output_tokens: int = 0,
    ) -> None:
        self.records.append({
            "model": model,
            "cost_usd": cost_usd,
            "input_tokens": input_tokens,
            "output_tokens": output_tokens,
            "timestamp": datetime.now(timezone.utc).isoformat(),
        })
        self.total_usd += cost_usd
        if self.total_usd > self.daily_budget:
            raise BudgetExceededError(
                f"日预算超限: ${self.total_usd:.4f} > ${self.daily_budget:.2f}"
            )

    def report(self) -> dict:
        by_model = {}
        for r in self.records:
            m = r["model"]
            by_model.setdefault(m, {"calls": 0, "cost_usd": 0.0})
            by_model[m]["calls"] += 1
            by_model[m]["cost_usd"] += r["cost_usd"]
        return {
            "total_usd": round(self.total_usd, 4),
            "budget_usd": self.daily_budget,
            "budget_used_pct": round(self.total_usd / self.daily_budget * 100, 2),
            "by_model": by_model,
            "call_count": len(self.records),
        }

    def save(self, path: Path) -> None:
        path.write_text(json.dumps(self.report(), indent=2, ensure_ascii=False))


class BudgetExceededError(Exception):
    """AI 日预算超限异常。"""
    pass
```

```python
def quick_neutrality_check(text: str) -> dict:
    """快速关键词扫描，无需 AI 调用。

    Returns:
        {"score": 0-1, "issues": [...]}
    """
    issues = []
    for pattern, reason in BIAS_PATTERNS:

        if matches:
            issues.append({
                "pattern": pattern,
                "reason": reason,
                "count": len(matches),
            })

    score = max(0.0, 1.0 - 0.1 * len(issues))
    return {"score": score, "issues": issues}
```

```python
# scripts/ai/cache.py
"""AI 响应缓存，相同 prompt 不重复调用。"""
import hashlib
import json
from pathlib import Path

CACHE_DIR = Path("/data/ai_cache")
CACHE_DIR.mkdir(parents=True, exist_ok=True)


def cache_key(prompt: str, model: str) -> str:
    """生成缓存键。"""
    content = f"{model}:{prompt}"
    return hashlib.sha256(content.encode()).hexdigest()


def get_cached_response(prompt: str, model: str) -> str | None:
    """获取缓存响应。"""
    key = cache_key(prompt, model)
    cache_file = CACHE_DIR / f"{key}.json"
    if cache_file.exists():
        data = json.loads(cache_file.read_text())
        return data["response"]
    return None


def set_cached_response(prompt: str, model: str, response: str) -> None:
    """保存缓存响应。"""
    key = cache_key(prompt, model)
    cache_file = CACHE_DIR / f"{key}.json"
    cache_file.write_text(json.dumps({
        "model": model,
        "prompt_hash": key,
        "response": response,
    }, ensure_ascii=False))
```

### 8.3 降级链实现

```python
# scripts/ai/fallback.py
"""模型失败降级链。"""
import logging

from ai.clients import ClaudeClient, GPT4oClient, GeminiClient

log = logging.getLogger(__name__)


def chat_with_fallback(
    prompt: str,
    primary: str,
    cost_tracker,
    json_mode: bool = False,
) -> str:
    """调用主模型，失败时按降级链尝试。

    Args:
        prompt: 提示词
        primary: 主模型名（claude/gpt4o/gemini）
        cost_tracker: 成本追踪器
        json_mode: 是否 JSON 模式

    Returns:
        模型响应文本
    """
    fallback_chain = {
        "claude": [
            ("claude", ClaudeClient),
            ("gpt4o", GPT4oClient),
            ("gemini-pro", None),  # Gemini 1.5 Pro
        ],
        "gpt4o": [
            ("gpt4o", GPT4oClient),
            ("claude", ClaudeClient),
            ("gemini-flash", GeminiClient),
        ],
        "gemini": [
            ("gemini-flash", GeminiClient),
            ("gpt4o-mini", None),  # GPT-4o mini
            (None, None),  # 跳过
        ],
    }

    chain = fallback_chain.get(primary, [("gpt4o", GPT4oClient)])
    for model_name, client_cls in chain:
        if client_cls is None:
            log.warning(f"[FALLBACK] {model_name} 未配置，跳过")
            continue
        try:
            client = client_cls(cost_tracker=cost_tracker)
            return client.chat(prompt, json_mode=json_mode)
        except Exception as e:
            log.warning(f"[FALLBACK] {model_name} 失败: {e}，尝试下一个")
            continue

    raise RuntimeError(f"所有降级模型均失败，primary={primary}")
```

```python
def calculate_quality_score(audit_result: dict) -> float:
    """计算综合质量分。"""
    weights = {
        "originality": 0.25,
        "neutrality": 0.30,
        "factual": 0.25,
        "fluency": 0.10,
        "seo": 0.10,
    }
    score = (
        audit_result["originality"]["score"] * weights["originality"]
        + audit_result["neutrality"]["score"] * weights["neutrality"]
        + audit_result["factual"]["score"] * weights["factual"]
        + audit_result.get("fluency", {}).get("score", 0.8) * weights["fluency"]
        + (1.0 if audit_result.get("seo", {}).get("passed", True) else 0.0) * weights["seo"]
    )
    return round(score, 3)
```

```python
# scripts/ai/ab_test.py
"""模型 A/B 测试：对新旧模型同时调用，对比质量分。"""
import json
from pathlib import Path

from ai.clients import ClaudeClient
from ai.neutrality_scorer import ai_neutrality_score


def ab_test_model(
    prompt: str,
    old_model: str,
    new_model: str,
    test_cases: list[dict],
) -> dict:
    """对新旧模型进行 A/B 测试。

    Returns:
        {
            "old_model": {"avg_score": 0.85, "avg_cost": 0.01, "avg_latency": 2.5},
            "new_model": {"avg_score": 0.88, "avg_cost": 0.008, "avg_latency": 2.0},
            "recommendation": "upgrade/keep/rollback"
        }
    """
    results = {"old_model": [], "new_model": []}

    for case in test_cases:
        # 调用旧模型
        old_client = ClaudeClient(model=old_model)
        old_response = old_client.chat(prompt.format(**case))
        old_score = ai_neutrality_score(json.loads(old_response), case.get("country", "us"), old_client)

        # 调用新模型
        new_client = ClaudeClient(model=new_model)
        new_response = new_client.chat(prompt.format(**case))
        new_score = ai_neutrality_score(json.loads(new_response), case.get("country", "us"), new_client)

        results["old_model"].append(old_score["score"])
        results["new_model"].append(new_score["score"])

    old_avg = sum(results["old_model"]) / len(results["old_model"])
    new_avg = sum(results["new_model"]) / len(results["new_model"])

    recommendation = "upgrade" if new_avg > old_avg + 0.02 else "keep"

    return {
        "old_model": {"avg_score": round(old_avg, 3)},
        "new_model": {"avg_score": round(new_avg, 3)},
        "recommendation": recommendation,
    }
```
